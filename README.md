# Smart Weather Intelligence System

## Description
The Smart Weather Intelligence System analyzes historical weather data to provide insights, trends, and summaries using Python. It uses Pandas for data handling, NumPy for calculations, and Matplotlib for visualizations. This system helps users understand weather patterns, detect extreme conditions, and supports basic decision-making.

## Features
- Load and clean weather datasets
- Display temperature, humidity, rainfall trends
- Generate visualizations for analysis
- Basic rule-based alerts (e.g., extreme temperatures)

## Technologies Used
- Python 3.x
- Pandas
- NumPy
- Matplotlib

## How to Run
1. Clone the repository: `git clone <repo_url>`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the main file: `python app.py` or `python main.py`
